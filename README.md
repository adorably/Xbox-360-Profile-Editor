# Xbox-360-Profile-Editor

Required:
- Python 3.10

` pip install httpx `

*This is being posted for the sake of keeping random people safe in the Xbox community who have been getting their accounts hijacked from unsafe bots. These bots are typically ran by people who are known scammers, but always end up finding more victims.*

*The original gamerpicture data set was found in July of 2022, and shared with me by vv. I've added all profile content sets to this tool, and decided to release this publicly to try and help reduce the amount of account hijackings.*
